# Calculator Web App

## Objective
The objective of this project is to create a fully functional calculator using HTML, CSS, and JavaScript.
This project focuses on building an interactive user interface and implementing basic mathematical operations.

## Tools & Technologies Used
- HTML – Structure of the calculator
- CSS – Styling and layout using CSS Grid
- JavaScript – Logic and interactivity
- VS Code – Code editor
- Web Browser – Testing and execution

## Features
- Perform basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
- Clean and modern calculator UI
- Responsive button grid using CSS Grid
- Clear (C) and Delete (DEL) functionality
- Real-time input and result display

## Steps Performed
- Designed calculator layout using HTML
- Used CSS Grid for button alignment
- Styled the calculator with modern UI effects
- Implemented calculator logic using JavaScript
- Added event listeners for interactive functionality

## Outcome
The final result is a visually appealing and fully functional calculator web application.
This project enhanced my understanding of DOM manipulation, event handling, CSS Grid layout, and JavaScript logic.
